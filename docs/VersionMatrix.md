# SharePoint Framework to `spfx-fast-serve-helpers` version matrix

| SPFx version      | `spfx-fast-serve-helpers` version                                   |
|-------------------|-------------------------------------------------------------------|
| < 1.4.1           | not supported                                                     |
| [1.4.1 - 1.8]       | ~1.4.0                                                            |
| [1.9 - 1.11]        | ~1.11.0                                                           |
| [1.12 - the latest] | ~[SPFx version].0, i.e. for SPFx 1.14 the version will be ~1.14.0 |
